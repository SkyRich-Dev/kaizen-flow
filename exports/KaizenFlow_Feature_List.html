Feature List Document

Version: 1.0 | Date: December 2025

1. User Management Features

1.1 Authentication & Authorization

  • Secure JWT-based user authentication
  • Role-based access control (RBAC)
  • Session management with automatic token refresh
  • Password-protected user accounts

2. Kaizen Request Management

2.1 Request Creation

  • Comprehensive request form with all required fields
  • Station/line/program identification
  • Issue description and poka-yoke details
  • Cost estimation with currency support (INR)
  • Expected benefits selection (multiple choice)
  • Effect of changes documentation
  • File attachment support (PFMEA, Check Sheets, CRR, Photos)
  • Draft saving capability

2.2 Request Tracking

  • Unique request ID generation (KZN-YYYY-NNNN)
  • Real-time status tracking
  • Complete audit trail
  • Request history and timeline view
  • Filtering and search capabilities

3. Approval Workflow Features

3.1 Multi-Stage Approval Process

StageApproverAction Required
1. DraftInitiatorSubmit request
2. Own ManagerDepartment ManagerReview & approve/reject
3. Own HODDepartment HODDepartment approval
4. Cross-ManagersAll other dept ManagersRisk evaluation (4 approvals needed)
5. Cross-HODsAll other dept HODsRisk evaluation (4 approvals needed)
6. AGMAGMRequired if cost >₹50k or process/manpower additions
7. GMGMRequired if cost >₹100k
8. FinalSystemApproved/Rejected status

3.2 Department Risk Evaluation

  • Mandatory risk assessment checklists per department
  • 27 evaluation questions across 5 departments
  • Risk level rating (Low/Medium/High)
  • Mandatory remarks for "No" answers or high-risk items
  • Acceptance/rejection tracking per department

3.3 Cost-Based Routing

  • Automatic escalation based on cost thresholds
  • ₹0 - ₹50,000: HOD approval sufficient
  • ₹50,001 - ₹100,000: Requires AGM approval
  • Above ₹100,000: Requires GM approval
  • Process/manpower additions: Mandatory AGM escalation
